# Campaign 2 · Ф1 — sessions: хребет атрибуции

_2026-07-10, ~12:00–12:27. Коммиты: `1c9757b` (feat), `ea6ed83`
(ledger). Floor: all green (118 тестов, conform 0, specmap 18/52/0)._

## Что сделано

MC получил измерение «босс-сессия»: `SessionId` (ULID),
`SessionRecord` со счётчиками инициативы, **отдельный**
`sessions.jsonl` с собственным fold-ом (журнал ранов не тронут ни
байтом), идемпотентный begin по `(harness, external_id)`, канал
фактов (`work_tool`/`delegated`/`nudge_sent`/`question_alert`),
сессионные метрики (бакет ранов + паркованные вопросы с возрастами).
Раны несут `origin_session`; CLI штампует его из
`FRACTALITY_BOSS_SESSION`; `session begin|end|show|ls` в CLI;
I1-тест пинует, что переменная не попадает воркерам. Интеграционные
тесты (7 штук) написал делегат GLM-5.2 по прецизионному work-order-у
— и прошёл их сам; я перепроверил своим прогоном.

## Решения (главное)

- **Второй журнал, не расширение первого.** У `Event::run_id()`
  тотальная сигнатура; впихивать session-события в тот же enum —
  ломать реплей ради ложной общности. Писатель/реплей обобщены по
  «stem» — вся ротация и torn-tail-толерантность переиспользованы.
- **Атрибуция — метка, не ссылка.** Битый/чужой `origin_session`
  никогда не валит регистрацию; Delegated-нота на неизвестную сессию
  дропается с warn. Facts, not references.
- **BD1 «чистый лист»:** Delegated-нота обнуляет
  `work_tools_since_delegation`, история (`total`) живёт. Психология
  барка́ина, механика наша.
- **Один feat-коммит вместо двух плановых** — поле + подсистема +
  CLI-поверхность компилируются только вместе; искусственный шов
  ради церемонии = некомпилирующийся промежуточный коммит. Дрейф
  заледжерен (P7-бухгалтерия).
- **Политика НЕ в MC** (D3 подтверждён кодом): MC копит факты;
  пороги/тексты — в движке. Демон не редеплоится ради tuning-а
  промптов.

## Идеи и размышления

- `session begin` как idempotent-lookup оказался элегантнее
  отдельного «find by external id» эндпойнта — API не растёт, а
  resume/clear/compact Клода естественно садятся на одну запись.
- Делегат снова показал класс на шаблонных тестах (7/7 с первой
  посадки, стиль дома выдержан). Экономика: мой один work-order
  против ~300 строк тестов.

## Косяки и висяки

- **`session ls` без лимита/фильтра по дате** — на живом
  `~/.fractality` через месяц будет простыня. Нужен `--limit`/`--open`
  по умолчанию? (Пока `--open` есть, дефолт — всё.)
- **`cwd`-фоллбек атрибуции из D2 (α) НЕ реализован** — только env.
  Осознанно: env-путь закрывает всё, что нужно трайлу; фоллбек
  добавим, если полевые данные покажут дыру. Записано здесь, чтобы
  не потерялось.
- **Сессии не умирают сами**: краш харнесса без SessionEnd оставит
  вечно-open сессию. Идея: TTL/`last_seen` + жатва по heartbeat-ам
  хуков — стратегический пункт, не в этой кампании.
- Python-скрипт в отчётной сессии дважды спотыкался о heredoc/квоты —
  мой косяк, не делегата; Edit-инструменты надёжнее шелл-магии
  (совпадает с машинным законом правок).
