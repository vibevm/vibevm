The whole loop is real: `fractality gate` decides, `spawn` opens a child
packet, mission-control admits it (depth-guard, sibling invariants,
cold-verifier check), a pod launches a real GLM Claude-Code worker, the
worker works and writes a result, acceptance runs, `Collected` folds onto
the bus, and — where the task cannot be finished here — `escalate` hands it
UP the tree. Descent and ascent, both wired, both tested, both observed
live.

## The decisions that shaped Stage B (long form)

- **Escalation is a terminal outcome, expressed by an MCP tool** (Ф4): the
  worker is `running` when it decides, so `running → escalated` is clean; a
  result-status exit would fight the state machine. The record climbs the
  `parent` edges; the run does not resume.
- **Verifier-accept is a readable verdict, not a hard completion gate**
  (Ф5): §10.8 conservative — the verdict is first-class on the record, but
  it does not suspend the tree.
- **The single-arm, smoke-test-first trial** (Ф6): one gated arm vs the C2
  naive baseline halved the spend; run 1 proved the system ran end to end
  before the rest of the ≤8-session cap was committed.
- **The delegation mechanism switched opencode → CC+z.ai** (mid-session,
  owner-prompted): headless Claude Code at the z.ai gateway — fractality's
  own worker mechanism. It unblocked this session's grunt delegation AND is
  the exact mechanism the trial arms run on. The pilot ate its own dog
  food, end to end.

## Косяки и висяки (honest, all filed)

- **Worker turn caps dominated completion** — `delegated` (8) is honest,
  but `delegated-AND-collected` (3) trailed it purely because workers timed
  out at 30 turns. Config, not fabric. → PP-004.
- **Two predictions unexercised** (schema gate, Silo escalation) — menu
  gaps. → PP-004.
- **No `fractality decisions` read verb** — the gate journals decisions but
  nothing reads them back, so P-C3-a is a direction not a number. → PP-004.
- **Worker-stop after escalate is cooperative, not enforced** (Ф4); **no
  tree→verifier query** (Ф5); **acceptance does not feed routing yet**
  (FD-5 soft-label table, off D-C3-8) — all non-blocking, all noted in
  their phase reports.

## Strategic verdict

Stage B set out to build a recursive-language-model fabric and prove it
runs. It did both. The strongest evidence is not the 44.4% — it is that
the number exists at all: a cold boss under a cheap model, handed the
fractality fabric and told to gate its work, actually delegated to workers
that actually ran and actually returned results, through files on disk,
under real pods, live. The design is sound. What remains is tuning (caps,
menus, a couple of read verbs) and the deferred scope the owner parked:
**Option C — the advisor slice (PP-003)**, next.
