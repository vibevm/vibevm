# Campaign 2 · Ф2 — scoreboard: движок и первые потребители

_2026-07-10, ~12:28–12:37. Коммиты: `6f5788a` (движок), `6d8397e`
(глагол + роллап), `a979ea6` (ledger). Floor: all green (conform 0

## Что сделано

Седьмой крейт `fractality-initiative` — чистый движок
политики/рендера: ни I/O, ни часов, ни собственного состояния;
на вход DTO шины + явные таймстемпы, на выход строки и решения.
Ф2-половина: `render_line` (стрип для statusline/инъекций),
`render_board` (сессия → паркованные вопросы → today/month),
`month_web_calls` (месячный роллап квоты — закрыт хвост IGNITION
§15). CLI: `scoreboard [--session|--line|--json]` (выбор сессии:
флаг → env → global-only), `stats` печатает месячную строку.
Попутно main.rs перерос бюджет 600 строк — семейство `mc` уехало в
свою клетку `mc_cmd.rs` (гейт поймал, шов ответственности очевиден).

## Решения (главное)

- **D7 «строго факты» — в коде, не в декларации.** Никаких
  «сэкономлено X босс-токенов»: методологии измерения нет — числа
  нет. Тест пинует точные строки борда; врать скорборд не умеет
  синтаксически.
- **Движок отдельным крейтом** (D8): пять юнит-тестов на строки —
  вся политика тестируется без демона, без сети, без харнесса.
  CLI/хуки — тонкие оболочки.
- **Часы снаружи:** `today`/`month`/`now_ms` — аргументы. Рендер
  детерминирован; goldens не флапают.
- **Ambient-выбор сессии в `scoreboard` — best-effort:** явный
  `--session` обязан резолвиться (опечатка = ошибка), env-переменная
  деградирует в global-борд молча (закон доступности).

## Идеи и размышления

- Скорборд-строка нарочно опускает нулевые факты (failed/parked) —
  меньше шума, аномалия видна мгновенно. Обратная сторона: строка
  «дышит» шириной; для statusline это ок.
- Роллап через `by_day`-префикс — дёшево и без новых измерений в MC.
  Когда день-бакетов станут тысячи, понадобится серверный агрегат —
  записано ниже.

## Косяки и висяки

- **`by_day` растёт неограниченно** — метрики считаются fold-ом по
  всем ранам на каждый запрос. На v0.1-объёмах — микросекунды; на
  годе телеметрии — пересчитать. Стратегически: инкрементальные
  агрегаты в MC (кэш по террминальным ранам) — отдельная кампания,
  не сейчас.
- **`scoreboard --line` без сессии** печатает today-строку — слегка
  другой словарь, чем сессионный стрип. Не унифицировано; мелочь,
  но глаз цепляет.
- Квота показывает только счётчик calls — без лимита плана
  (Lite 100 / Pro 1000 / Max 4000, D12). Чтобы показать «X из 4000»,
  нужен `profile.pricing.plan` в metrics-ответе — маленький хвост
  на Ф4/Ф6.
